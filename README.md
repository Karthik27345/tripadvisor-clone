# Tripadvisor Clone (Responsive Website)

This is a responsive Tripadvisor-inspired website built using **HTML and CSS**.

## Features
- Responsive design (Desktop, Tablet, Mobile)
- Flexbox-based layout
- Sticky search bar
- Card-based experience section
- Iconic places section
- Mobile-friendly footer
- Floating "Ask Ollie" button

## Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries

## Learning Outcome
This project helped me understand:
- Responsive layouts
- Mobile-first adjustments
- Debugging overflow issues
- Real-world UI structure
